### Downloading all the playgrounds

除非另有说明，否则所有 playground 都与 Swift 3 兼容.

所有 playgrounds 都可以作为 `playgrounds/` 目录中的子模块使用，要一次下载它们，只需使用 `git clone --recursive https://github.com/uraimo/Awesome-Swift-Playgrounds 克隆这个存储库. git` 或在以通常方式克隆存储库后执行 `git submodule update --init`. 

必须手动下载以 zip 存档形式分发的 Apple 游乐场.
