## Index


## Contribute 🌟

这是一个社区策划的列表，请提交您的 PR 以及您知道和喜欢的黑客攻击点！

&gt; 问：是什么让黑客点很棒？

一个你会去和你的朋友一起去黑客的地方，电源和 wifi 是一个加分项，喝咖啡和小吃是一个加分项. 我们现在排除需要订阅或某种会员资格的地方（除非会员资格可以在访问时完成并且不需要长期承诺）.

## Hackerspaces
