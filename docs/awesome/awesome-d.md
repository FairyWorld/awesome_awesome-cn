## Compilers

*从源代码编译软件.*

* [dmd](https://github.com/dlang/dmd)  - D 编程语言的参考编译器. 稳定，构建速度极快，非常适合学习和快速原型设计/开发. 目前前端是用D实现的，在dmd、ldc和gdc之间共享，后端是用C++实现的.
* [ldc](https://github.com/ldc-developers/ldc)  - 基于 LLVM 的 D 编译器. 使用 DMD 前端和 LLVM 后端. 构建速度比 dmd 慢，但生成的代码比 DMD 更优化. 它支持 LLVM 的所有目标平台.
* [gdc](https://github.com/D-Programming-GDC/GDC)  - GNU D 编译器. 使用 DMD 前端和 GCC 后端. 由于使用 GCC，目前针对大多数平台. 在大多数情况下，生成的代码比 DMD 运行得更快，与 LDC 相当. 在与官方 GCC 工具链集成的过程中.
