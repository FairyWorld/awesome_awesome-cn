精选的令人愉快的清单 [PowerShell](https://en.wikipedia.org/wiki/PowerShell) 包和资源.

PowerShell 是一种跨平台（Windows、Linux 和 macOS）自动化和配置工具，针对处理结构化数据（例如 JSON、CSV、XML 等）、REST API 和对象模型进行了优化.
它包括一个命令行 shell 和一个相关的脚本语言.
