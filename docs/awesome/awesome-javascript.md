## Package Managers
*托管 JavaScript 库并提供获取和打包它们的工具.*

* [npm](https://www.npmjs.com/) - npm 是 JavaScript 的包管理器.
* [Bower](https://github.com/bower/bower) - 网络包管理器.
* [component](https://github.com/componentjs/component) - 用于构建更好的 Web 应用程序的客户端包管理.
