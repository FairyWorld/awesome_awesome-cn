- [slack-ruby-client](https://github.com/dblock/slack-ruby-client) - 用于 Slack Web 和实时消息传递 API 的 Ruby 客户端

### Rust

- [slack-morphism-rust](https://github.com/abdolence/slack-morphism-rust) - 用于 Rust 的现代异步 Slack API 客户端
